What is Focusly?

Focusly lets you move around complex, data-heavy UIs using only the keyboard. Use Alt + Arrow Keys to move between interactive cells, and press Enter to trigger the contextual action for that panel.

Component Toolkit Independence

Focusly is designed to work seamlessly across all UI component toolkits. In this demo you can switch between Vanilla HTML, Ng-Zorro and Angular Material. Toolkit-specific support is implemented as a thin extension of the base directive, meaning the core navigation logic stays centralised and clean while each UI library only adds a few small, tailored overrides. This keeps behaviour consistent, predictable, and incredibly easy to maintain.
