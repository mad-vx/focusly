New FocuslyShortcutHost directive. This allows scoping of shotcut keys to a parent element. It allows UI elements that do not implement the focusly directive to participate in the shotcut key hierarchy. This is a significant step towards establishing a unified apporach to keyboard navigation and shotcuts all under one framework that can work together seemlessly.
